# 🏥 Hospital Appointment Booking System

## Project Overview
The Hospital Appointment Booking System is a web-based application that allows patients to book appointments with doctors online. It helps reduce waiting time, improves appointment management, and provides an easy way for hospitals to manage patient records.

## Features
- Patient Registration and Login
- Doctor Registration and Login
- View Available Doctors
- Book Appointments
- Cancel Appointments
- Appointment History
- Admin Dashboard
- Manage Doctors
- Manage Patients
- Manage Appointments

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: XAMPP

## Modules

### Patient Module

## Installation

1. Install XAMPP.
2. Copy the project folder into the **htdocs** directory.
3. Start Apache and MySQL from the XAMPP Control Panel.
4. Create a MySQL database.
5. Import the SQL file into the database.
6. Open your browser and visit:

```
http://localhost/Hospital-Appointment-Booking-System/
```
